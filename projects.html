Daily Grind - Coffee Shop - (code)

Daily Grind is a client side application for a coffee shop opening in downtown manhattan Summer of 2015. Clients can sign up and manage their subscription via the website and use their barcode to get unlimited coffee. Once a barcode is recieved, the in-store app will make an API call to this app via #/find/:barcode to return a JSON object and verify subscription status.

Built with: Ruby on Rails, PostgreSQL, Heroku, bootstrap, devise, Stripe API, bwip-js barcode API
Status: in progress
License: Apache
